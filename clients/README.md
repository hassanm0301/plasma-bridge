# Clients

This folder is reserved for client applications that consume the `plasma_bridge` backend APIs.

- [web](web/): React web dashboard
- [app](app/): planned Flutter mobile app

The project version is managed once at the repository root in [../VERSION](../VERSION), and release notes live in [../CHANGELOG.md](../CHANGELOG.md).

Today the implemented client is the React dashboard, which renders live windows, current media playback, and audio device state from the `plasma_bridge` backend.
